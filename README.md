# Math Quiz Python

## Overview

Math Quiz Python is a simple quiz application developed using Python.
The application generates mathematical questions, accepts user answers, and calculates the final score.

This project was created as a part of my Python learning journey. 
While working on this project, I improved my understanding of Python programming concepts,
strengthened my problem-solving skills, and explored the Tkinter module for creating interactive applications.

## Features

- Generates random mathematical questions
- Accepts user input and checks answers
- Calculates and displays the final score
- Interactive user interface using Tkinter
- Simple and beginner-friendly implementation

## Technologies Used

- Python 3
- Tkinter
- Random module

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/Harshithasreedhar5/maths-quiz-python.git
```

2. Navigate to the project folder:

```bash
cd Math-Quiz-Python
```

3. Run the Python file:

```bash
python math_quiz.py
```
